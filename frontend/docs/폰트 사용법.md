## Font Usage

기본 폰트는 `Pretendard Regular`입니다.  
일반 텍스트는 따로 폰트를 지정하지 않아도 됩니다.



강조 텍스트는 아래 스타일을 지정하여 사용합니다.

```xml
style="@style/Text.Pretendard.SemiBold"(프리텐다드 세미볼드)
style="@style/Text.Pretendard.Bold"(프리텐다드 볼드)
style="@style/Text.Baloo.Regular" (Baloo 일반)
style="@style/Text.Baloo.Bold" (Baloo 볼드)
```

예시:

```xml
<TextView
    style="@style/Text.Pretendard.Bold"
    android:text="설정" />
```

```xml
<TextView
    style="@style/Text.Baloo.Bold"
    android:text="Chat" />
```

사용 기준:

```text
기본 텍스트 → Pretendard Regular
중간 강조 → Pretendard SemiBold
강한 강조/제목 → Pretendard Bold
영문 로고/타이틀 → Baloo Bold
```

Baloo는 영어 텍스트에만 사용합니다.
